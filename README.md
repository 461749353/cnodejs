# 上传文件方法


```shell

# 1.使用git clone命令将新建的git 项目信息下载到本地
git clone https://github.com/4614749353/cnodejs.git cnodejs

# 切换到 ithub 目录 进入git项目的本地文件夹，将自己的项目复制到这个git 文件夹，查看状态，提交所有的新文件
cd ithub        
git status  查看全部文件
git add -A  上传的文件 -A可改成单个文件 ./文件名
git commit -a -m"上传备注文字信息"  上传单个文件 -a 忽略

# 上传更新到git 版本库
git push  

```
